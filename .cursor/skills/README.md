# Project Agent Skills (Cursor)

Curated copies from **[Antigravity Awesome Skills](https://github.com/sickn33/antigravity-awesome-skills)** (MIT) for this repo’s stack: **Next.js App Router, React, Tailwind, APIs, Supabase**.

## Bundled skills

| Folder | Use when |
|--------|----------|
| `frontend-design` | UI polish, distinctive layouts, avoid generic “AI UI” |
| `react-best-practices` | Components, rerenders, bundles, async patterns |
| `tailwind-patterns` | Utility-first structure |
| `testing-patterns` | Tests and validation |
| `api-design-principles` | Route and API shape |
| `nextjs-app-router-patterns` | App Router conventions |
| `nextjs-best-practices` | Next.js-specific guidance |
| `supabase-automation` | Supabase workflows |

## How to use

In Cursor chat, invoke naturally, e.g. *“using **frontend-design**, improve this dashboard card”* or *“apply **nextjs-app-router-patterns** to this route”*.

## Add more from the same library

From the **Cyberhound** repo root (optional):

```bash
npx antigravity-awesome-skills --cursor --path .cursor/skills
```

Or copy individual folders from your clone: `../antigravity-awesome-skills/skills/<name>/`.
